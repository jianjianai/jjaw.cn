配置文源基础

所有的配置文件都在 vuepress.config.ts 文件中，你可以根据自己的需求进行修改。

所有有关网盘的配置都在 FileList 函数的参数中。FileList 接收一个文件数组，可以配置挂载多个文件源。

注意，每个对象都要用{}包裹，每个对象之间用,隔开。 例：

export default defineUserConfig({
    ....
    theme: FileList([
        {...},
        {...},
        {...},
        {...}
    ]),
    .....
});
